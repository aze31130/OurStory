# Custom potions

## Potion of Resistance
- **Description:** Increases your resistance for 5 minutes
- **Craft:**
    ```
    +---+---+---+
    |   | T |   |
    +---+---+---+
    |   | P |   |
    +---+---+---+
    |   |   |   |
    +---+---+---+

    P = Thick Potion
    T = Turtle Egg
    ```
